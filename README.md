# Simulateur de Revenu d'Intégration - CPASConnect

## Intégration via iframe

Pour intégrer ce simulateur dans votre site web, utilisez le code suivant :

```html
<iframe 
    src="https://suyttenhoef-cyber.github.io/simulateur-ri/embedded.html" 
    width="100%" 
    height="900px" 
    frameborder="0" 
    style="border-radius: 15px; border: 2px solid #E1E8ED; box-shadow: 0 8px 32px rgba(0, 0, 0, 0.08);"
    title="Simulateur de Revenu d'Intégration"
    allow="clipboard-write; encrypted-media;"
    sandbox="allow-scripts allow-same-origin allow-forms allow-popups allow-modals"
></iframe>
